# Image Slideshow Effect with HTML & CSS

Um projeto simples de **slideshow de imagens** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar um **efeito de slideshow automático**, ideal para galerias de imagens e banners.

## 🛠️ Tecnologias

- HTML
- CSS (animation, keyframes)

## 🎯 Funcionalidades

- Slideshow automático
- Transição suave entre imagens
- Design simples e moderno
- Sem uso de JavaScript

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
